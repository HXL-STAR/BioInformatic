4.i为整型变量，则以下循环语句的循环次数是

```c_cpp
for(i=2;i<2;) 
printf(“%d”,i--);
```

解析：**<u>0次</u>**

for语句的执行流程描述如下：
（1） 计算表达式1的值；
（2） 计算表达式2的值，若表达式的值为真（或非0），则执行循环体；
（3） 计算表达式3的值；
（4） 再次计算表达式2的值，并重复以上过程；
（5） 直到表达式2的值为假（或为0），则结束该循环。
