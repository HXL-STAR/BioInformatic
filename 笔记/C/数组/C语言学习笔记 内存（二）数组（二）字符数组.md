字符型数据是以字符的ASII代码存储在存储单元中的，一般占一个字节。

C语言字符串是存放在字符型数组中的。

### 定义

字符数组中一个元素存放一个字符，而且因为字符是ASII码形式存储所以可以用整型数组存储字符。

```c_cpp
char a[5];
a[0]='i';
a[1]='a';

int c[10];
c[0] = 'b';
```

### 初始化

- 最简单的方式是用“初始化列表”把各个字符依次赋给数组中元素
- 也可以用字符串常量对数组直接赋值
- 对字符数组赋值如果元素个数小于数组大小，其余位置赋值“\0”;
- 如果确定字符个数可以省略规定元素个数

### 处理

 C语言编译系统提供了一些处理字符串的库函数

puts 输出字符串

gets 输入字符串

strcat 字符串连接

strcp 字符串复制

strcmp 字符串比较

strlen 测试字符串长度
